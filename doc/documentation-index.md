# Homelab Dashboard Documentation

This folder contains the up-to-date project documentation in English.

## Documentation Index

- `user_doc/user-guide.md`  
  End-user guide: setup, navigation, day-to-day usage, and troubleshooting.

- `developer_doc/developer-guide.md`  
  Developer guide: architecture, local development, data model, and implementation notes.

- `developer_doc/api-reference.md`  
  Backend API reference for authentication, users, infrastructure, and settings.

## Current Project Status Snapshot

- Frontend: React + TypeScript + Vite + HeroUI (`frontend/`)
- Backend: Node.js + Express + Prisma + PostgreSQL (`backend-node/`)
- Python backend: reserved but currently empty (`backend-python/`)
- Container orchestration: `docker-compose.yml` at repository root
- Authentication: JWT-based login/setup flow
- Major implemented domain area: Documentation module (Overview, Hardware, Services, Storage, Markdown Docs, Map)

For quick start and architecture details, continue with the user and developer docs above.
