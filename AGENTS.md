# AGENTS.md

## Purpose
This file provides essential instructions and conventions for AI coding agents working in this repository. All code and comments must be written in English.

## Key Conventions
- **Language**: All code, comments, and documentation contributions must be in English. Do not use German or any other language for code comments or documentation.
- **Frontend**: Located in `frontend/` (React + TypeScript + Vite). See `frontend/README.md` for build and linting details.
- **Backend**: Located in `backend-node/` (Node.js + Express + Prisma). See root `README.md` and `doc/developer_doc/developer-guide.md` for architecture and API details.
- **Documentation**: All technical and user documentation is in `doc/`. See `doc/documentation-index.md` for an overview.
- **Repository Structure**: Refer to `doc/developer_doc/repository-structure.md` for the up-to-date file/folder layout.

## Build & Development
- Use Docker Compose for full-stack local development (`docker-compose.yml`).
- For frontend, use Vite commands as described in `frontend/README.md`.
- For backend, use Node.js scripts as described in `backend-node/`.

## Additional Notes
- If you add new documentation or code, ensure it is in English and update relevant links in this file if needed.
- When in doubt, prefer linking to existing documentation rather than duplicating content.
